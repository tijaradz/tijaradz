# 🤖 SEO Health Bot

Automatic SEO monitoring on Netlify.

## Quick Start

1. `npm install`
2. Create `.env.local` from `.env.example`
3. `netlify functions:invoke seo-monitor`
4. Push to GitHub
5. Deploy on Netlify
6. Add env variables in dashboard

## Features

✅ HTTP Status Check
✅ JSON-LD Detection
✅ Meta Tags Check
✅ Health Score (0-100%)
✅ Discord/Telegram Alerts
✅ Automatic 24h Monitoring
